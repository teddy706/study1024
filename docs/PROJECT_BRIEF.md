# 프로젝트 개요 (Project Brief)
**업데이트: 2025년 10월 28일**

## 1. 프로젝트 정보

### 프로젝트명
영업사원의 개인 비서 (AI-Powered Sales Assistant)

### 프로젝트 목표
영업사원의 일상적인 업무를 자동화하여 생산성을 향상시키고, 더 나은 고객 관리를 지원하는 AI 기반 웹 애플리케이션 개발

### 현재 상태
- **개발 진행률**: 약 70% 완료
- **핵심 기능**: 8개 중 8개 구현 완료 ✅
- **배포 상태**: 개발 환경에서 테스트 중

## 2. 비즈니스 가치

### 핵심 가치 제안
- 일일 데이터 입력 시간 90% 절감
- 고객 관리 효율성 200% 향상
- 영업 기회 발굴 시간 단축
- 실시간 고객 정보 접근성 개선

### 시장 기회
- 기업 영업사원: 약 50만명
- 공공기관 영업 담당자: 약 10만명
- 프리랜서 세일즈 전문가: 약 5만명

## 3. 핵심 기능

### ✅ 구현 완료
1. **명함 자동 등록 및 고객DB 구축**
   - Google Vision AI 기반 OCR
   - AI 기반 전화번호 자동 분류 (휴대폰/사무실/팩스)
   - 명함 이미지 저장 및 조회
   - 관심사 관리

2. **AI 조직 동향 리포트**
   - OpenAI GPT-4o 기반 분석
   - 사용자별 관심 조직 관리
   - 커스터마이징 가능한 프롬프트

3. **스몰토크 소재 자동 생성**
   - 고객 관심사 기반 AI 대화 소재
   - 자동 만료 및 캐싱
   - 출처 뱃지 (AI/Template)

4. **미팅 기록 관리**
   - 날짜 자동 기록
   - 메모 작성 (최대 300자)
   - 페이지네이션 (3개씩)
   - 댓글 스타일 UI

5. **통합 정보 대시보드**
   - 연락처 목록 및 검색
   - 리포트 조회
   - 연락처 상세 정보

6. **연락처 관리**
   - 전체 필드 편집 가능
   - 관심사 별도 관리
   - 명함 이미지 연동
   - 안전한 삭제

### 🔄 개발 진행 중
- Edge Function 배포 (명함 OCR)
- 통합 테스트

### 📋 향후 계획
- 통화 STT 기능
- 일정 관리 자동화
- 알림 시스템
- 모바일 앱

## 6. 기술 스택

### Frontend
- React 18 + TypeScript
- Vite (빌드 도구)
- TailwindCSS (스타일링)
- React Router (라우팅)

### Backend
- Supabase
  - PostgreSQL (데이터베이스)
  - Auth (인증)
  - Edge Functions (서버리스)
  - Storage (파일 저장)
  - Row Level Security

### AI/ML
- Google Vision AI (명함 OCR)
- OpenAI GPT-4o (텍스트 분석 및 생성)

### 배포
- Frontend: Vercel/Netlify
- Backend: Supabase Cloud
